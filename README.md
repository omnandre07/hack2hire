# Hack2Hire: AI-Powered Mock Interview Platform

Built for the **UnsaidTalks Hack2Hire Hackathon** on Unstop.

## Features
- Resume (PDF) and Job Description analysis
- Adaptive beginner-friendly questions (technical, conceptual, behavioral, project-based)
- Dynamic difficulty levels (Very Easy → Easy → Medium)
- Strict 2-minute time limit per question with overtime penalty
- Objective scoring based on accuracy, clarity, depth, relevance, and time efficiency
- Early termination for low performance
- Final readiness score (0–100), category (Strong / Average / Needs Improvement), detailed breakdown, and mentor feedback with actionable tips
- AI voice output: click "Hear Question" to listen (browser TTS)

## How to Run Locally
1. Clone the repo
2. Create virtual environment
3. Install dependencies
4. Create `.env` file with your Groq API key
5. Run the app
  

## Notes
- Voice output uses browser Text-to-Speech (best in Chrome/Edge)
- Voice input records audio (transcription can be added with Whisper API)
